# Foodables

**A comprehensive culinary expansion mod for Minecraft 1.21.5**

[![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21.5-brightgreen)](https://minecraft.net/) [![Fabric API](https://img.shields.io/badge/Fabric API-Required-orange)](https://fabricmc.net/) [![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE.txt) [![Downloads](https://img.shields.io/badge/CurseForge-Download-purple)](https://www.curseforge.com/minecraft/mc-mods/foodables-fabric)

Transform your Minecraft culinary experience with **125+ new items**, advanced food processing systems, and immersive farming mechanics. Foodables brings a comprehensive vanilla-plus approach to food, farming, and cooking that seamlessly integrates with your Minecraft world.

***

## üçé Features

### Food & Culinary System

*   **53 Unique Food Items** - From simple fruits to complex multi-ingredient dishes
*   **Balanced Nutrition System** - Tiered progression from basic foods (2-3 hunger) to premium meals (12+ hunger)
*   **Advanced Food Processing** - Baking, drying, juicing, and multi-step crafting systems
*   **Smart Saturation Values** - Complex recipes reward players with better saturation efficiency
*   **Always-Edible Drinks** - Juices and beverages that can be consumed even when full

### Farming & Agriculture

*   **7 Fruit Trees** - Apple, Banana, Lemon, Lime, Mango, Orange, and Peppercorn trees with natural generation
*   **5 New Crops** - Tomato, Lettuce, Brown Onion, Red Onion, and Garlic with full growth cycles
*   **2 Berry Bushes** - Blueberry and Blackberry bushes that generate naturally in forest biomes
*   **Tree Harvesting** - Trees drop fruits when the leaves are breaking over time

### Advanced Processing Systems

*   **Drying Rack Network** - 10 wood variants (including new Pale Oak) for jerky production
*   **Food Preservation** - Create jerky and dried foods with enhanced saturation and storage benefits
*   **Multi-Stage Cooking** - Complex recipes requiring multiple ingredients and processing steps
*   **Tool Integration** - 5 specialized tools (Mortar & Pestle, Knife, Juicer, Whisk, Crusher) for advanced crafting

### World Generation Integration

*   **Natural Tree Generation** - Fruit trees spawn organically in appropriate biomes
*   **Berry Bush Clusters** - Discover berry patches throughout forest regions
*   **Salt Ore Deposits** - Underground salt mining for food preservation and seasoning
*   **Configurable Generation** - Toggle any world generation feature via in-game configuration

## üì¶ Installation

### Requirements

*   **Minecraft**: 1.21.5
*   **Mod Loader**: Fabric Loader 0.16.14+
*   **Fabric API**: Latest version for 1.21.5 ([Download](https://modrinth.com/mod/fabric-api))
*   **Java**: 21 or higher
*   **Cloth Config**: 18.0.0+ ([Download](https://modrinth.com/mod/cloth-config)) - Required for configuration GUI

### Step-by-Step Installation

1.  **Install Fabric Loader** - Download and run the [Fabric Installer](https://fabricmc.net/use/installer/)
2.  **Download Dependencies**:
    *   [Fabric API](https://modrinth.com/mod/fabric-api) (Required)
    *   [Cloth Config](https://modrinth.com/mod/cloth-config) (Required for config menu)
3.  **Download Foodables** - Get the latest version from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/foodables-fabric)
4.  **Install Mods** - Place all `.jar` files in your Minecraft `mods` folder
5.  **Launch Game** - Start Minecraft using the Fabric profile

### Recommended Companion Mods

*   **[REI (Roughly Enough Items)](https://modrinth.com/mod/rei)** - Essential for viewing Foodables recipes and drying rack processes
*   **[AppleSkin](https://modrinth.com/mod/appleskin)** - Enhanced hunger and saturation display to understand food values
*   **[Mod Menu](https://modrinth.com/mod/modmenu)** - Required for accessing Foodables configuration options

## üîß Troubleshooting

### Common Issues & Solutions

**üö® Game crashes on startup**

*   Verify Fabric API and Cloth Config are installed for MC 1.21.5
*   Ensure all mods match your Minecraft version exactly
*   Confirm Java 21+ is installed and selected in launcher
*   Check launcher logs for specific error messages

**üå≥ Trees or ores not generating in my world**

*   Check configuration settings in Mod Menu ‚Üí Foodables Config
*   Remember: only NEW chunks will have Foodables generation
*   Try exploring unexplored areas or create a new world for testing
*   Verify world generation is enabled in configuration

**üìñ Recipes not showing in REI/recipe viewers**

*   Ensure REI version is compatible with Minecraft 1.21.5
*   Restart Minecraft after installing recipe viewing mods
*   Check that Foodables is loading properly (items appear in creative menu)
*   Try refreshing REI with `Ctrl+O` in-game

**üñºÔ∏è Missing textures or broken item models**

*   Verify mod file downloaded completely and isn't corrupted
*   Check for conflicting resource packs that might override Foodables textures
*   Try reinstalling the mod with a fresh download
*   Ensure no other mods are conflicting with Foodables item registration

**‚ö° Performance issues or lag**

*   Consider disabling world generation features in heavily modded packs
*   Reduce render distance when exploring for the first time with Foodables
*   Check if other mods are conflicting with Foodables world generation
